# 🚦 Transportation Data Analysis & Simulation (NHTS + NGSIM)

## 📌 Project Overview  
This project analyzes transportation usage and driving behavior in the United States using two datasets:  
- **NHTS** – travel behavior and trip characteristics  
- **NGSIM** – detailed driving behavior and vehicle trajectories  

The goal is to visualize key trends and simulate vehicle behavior using the Intelligent Driver Model (IDM).

---

## 🎯 What This Project Does  
- Cleans and processes transportation data  
- Creates visualizations:
  - Bar chart  
  - Histogram  
  - Boxplot  
  - Time-series plots  
- Simulates vehicle movement using IDM  

---

## ▶️ User Guide - How to Run (Jupyter Notebook)

1. Open the `.ipynb` file in Jupyter Notebook or VS Code  
2. Make sure required libraries are installed (pandas, matplotlib, seaborn)  
3. Run each cell in order  
4. Graphs and results will display in the notebook
